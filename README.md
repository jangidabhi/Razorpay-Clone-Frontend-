# Razorpay Clone Frontend

Welcome to the frontend repository for a Razorpay clone project. This project aims to replicate the basic user interface and user experience of the Razorpay platform using only HTML and CSS for markup and styling.

## Features

- **User Authentication**: Users can sign up, log in, and log out securely.
- **Dashboard**: Users are greeted with an intuitive dashboard upon logging in, where they can easily manage transactions, payment methods, and account details.
- **Payment Integration**: Seamlessly initiate payments, track transaction history, and manage payment methods.
- **Responsive Design**: The frontend is built with a mobile-first approach, ensuring a consistent and engaging experience across various devices and screen sizes.

## Technologies Used

- **HTML**: Markup language for building the structure of web pages.
- **CSS**: Styling language for enhancing the visual presentation of web pages.
- **Flexbox/Grid**: Layout techniques for creating responsive and flexible designs.
